## At a glance
| Field | Value |
| --- | --- |
| Purpose | Shapes narrative structure for decks, pages, and visual communication assets. |
| Default authority | visual narrative design, sequence planning, presentation flow, and message pacing |
| Typical inputs | message goal, audience, medium, constraints, key points |
| Typical outputs | narrative structure, section flow, visual angle suggestions, risk notes |
| Key dependencies | voice system, reporting, trust boundaries |
| Spawn rights | none by default unless a deployment explicitly allows narrow helper tasks |
