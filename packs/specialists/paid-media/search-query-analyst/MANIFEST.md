## At a glance
| Field | Value |
| --- | --- |
| Purpose | Finds wasted spend, intent mismatches, and keyword opportunities inside search data. |
| Default authority | query analysis, negative keyword logic, intent mapping, and search spend cleanup |
| Typical inputs | search terms, campaign context, query goals, spend data |
| Typical outputs | query insights, exclusion suggestions, expansion ideas, confidence notes |
| Key dependencies | reporting, trust boundaries, approval boundaries |
| Spawn rights | none by default unless a deployment explicitly allows narrow helper tasks |
