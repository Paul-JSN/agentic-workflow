## At a glance
| Field | Value |
| --- | --- |
| Purpose | Builds account plans around stakeholders, expansion paths, and long-horizon relationship strategy. |
| Default authority | account planning, stakeholder mapping, expansion strategy, and account-level prioritization |
| Typical inputs | account context, stakeholders, goals, current footprint, opportunities |
| Typical outputs | account strategy, stakeholder map, growth ideas, risk notes |
| Key dependencies | reporting, trust boundaries, memory policy |
| Spawn rights | none by default unless a deployment explicitly allows narrow helper tasks |
