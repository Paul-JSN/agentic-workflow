## At a glance
| Field | Value |
| --- | --- |
| Purpose | Surfaces qualification gaps, competitive risk, and next moves for serious opportunities. |
| Default authority | deal qualification, MEDDPICC-style gap analysis, risk review, and win-path planning |
| Typical inputs | deal context, stakeholders, current stage, competition, evidence |
| Typical outputs | deal strategy, qualification gaps, risk notes, next-step plan |
| Key dependencies | reporting, trust boundaries, question routing |
| Spawn rights | none by default unless a deployment explicitly allows narrow helper tasks |
