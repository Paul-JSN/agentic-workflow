## At a glance
| Field | Value |
| --- | --- |
| Purpose | Keeps recurring financial tracking, spend visibility, and reporting structure orderly. |
| Default authority | spend tracking, financial summaries, recurring reporting, and budget visibility |
| Typical inputs | financial records, budget categories, reporting need, cadence |
| Typical outputs | finance summary, spend notes, variance callouts, next-step recommendation |
| Key dependencies | reporting, approval boundaries, memory policy |
| Spawn rights | none by default unless a deployment explicitly allows narrow helper tasks |
