# Minimal Starter Bundle

## Use when
- the operator wants the simplest possible system
- the work is direct and not deeply research-heavy
- the operator is new and needs low overhead

## Includes first
- `shared/VOICE_SYSTEM.md`
- `shared/ORCHESTRATION.md`
- `shared/APPROVAL_BOUNDARIES.md`
- `shared/QUESTION_ROUTING.md`
- `shared/REPORTING_CONTRACT.md`
- `shared/ESCALATE.md`
- `shared/PLACEHOLDERS.md`
- `packs/core/primary-agent/TEMPLATE.md`
- `packs/core/executor/TEMPLATE.md`
- `packs/core/verifier/TEMPLATE.md`

## Why this exists
This is the easiest first working configuration. It gives the operator one front door, one execution lane, and one explicit verification lane.

## Leave out for now
- planner
- researcher
- critic
- vertical roles

## Upgrade path
If tasks become more multi-step or error-prone, move next to `STANDARD.md`.
